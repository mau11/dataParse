# dataParse

### Setting up MySQL Database:
- Create a database named "challenge"

### Preparing Application to Run Locally
From root directory:
- Seed questions to db
``
$ node seed.js
``
- Transpile/build app:
``
$ webpack
``
- Start Server
``
$ npm start
``
### View Application
Navigate to:
```sh
http://localhost:8000
```
